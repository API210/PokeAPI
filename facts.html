Facts

  • Aims to combine data for easy use across multiple websites.
  • Not only includes names, but includes abilities, moves, and other information pertaining to Pokemon.
  • Data is gathered from Veekun, having an open source project containing csv data.
  • The website uses React Static.
  • Includes information across all Pokémon generations.
  • The API is free and open use.
  • Also includes information about in-game items, berries, and foods.
  • Includes information about locations in the game.
  • Includes further details about Pokémon growth rates, characteristics and other statistics.
  • Includes information about languages available for the games.
